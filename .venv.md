# 🐍 Python Virtual Environment (.venv) Cheat-Sheet

Ein kompakter Leitfaden zum Einrichten virtueller Umgebungen und zur Verwaltung von Abhängigkeiten (`requirements.txt`) über das Terminal.

---

## 1. Einrichten & Aktivieren (Einmalig pro Projekt)

Navigiere zuerst in dein Hauptverzeichnis (`cd /Pfad/zu/Sales_system`), bevor du die Befehle ausführst.

| Aktion | Befehl (Mac / Linux) | Befehl (Windows CMD) |
| :--- | :--- | :--- |
| **`.venv` erstellen** | `python3 -m venv .venv` | `python -m venv .venv` |
| **`.venv` aktivieren** | `source .venv/bin/activate` | `.venv\Scripts\activate` |
| **`.venv` deaktivieren**| `deactivate` | `deactivate` |

> 💡 **Hinweis:** Sobald die Umgebung aktiv ist, siehst du ein präfixartiges `(.venv)` ganz links in deiner Terminal-Eingabezeile.

---
